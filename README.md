# DevCenterX

Panel web para crear, editar y publicar apps con IA.

## Qué contiene

- `index.html`: dashboard principal.
- `style.css`: diseño y responsive.
- `script.js`: interacción y navegación.
- `api/`: scripts del backend.
- Carpetas de secciones: `Apps/`, `Usage/`, `Published-apps/`, `Ajustes/`, `login/`, etc.

## Cómo usar

1. Abrir `index.html` en un navegador.
2. La app funciona como sitio estático.
3. Para producción, hospedar en Vercel u otro hosting.

## Recursos

- Usa CDN para `font-awesome`, `Chart.js`, `ExcelJS`, `PptxGenJS` y `docx`.
- `style.css` incluye breakpoints y ajustes para monitores grandes.
- `script.js` controla el comportamiento dinámico.

## Recomendación

- Probar en pantallas amplias y móviles.
- Mantener la estructura de carpetas y nombres.
